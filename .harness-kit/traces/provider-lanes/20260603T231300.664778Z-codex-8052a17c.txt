# /critique

Single-lens signal. Not a merge gate, not `/code-review`, not a parallel bench.

## Contract

- `--lenses` lists available lens names by reading
  `harnesses/shared/references/lenses.md`; do not hardcode the list here.
- `--lens <name>` resolves the matching `## <name>` block from that rubric.
  Unknown lens means stop and print the available names.
- `--target <path>` is required for a focused critique. Resolve it against the
  invoking repo and read only the files needed to answer.
- Never reference `agents/<lens>.md`. Lenses are rubrics, not static persona
  files.
- The critic is read-only. Do not edit, stage, commit, or emit a Ship verdict.

## Dispatch

Spawn one fresh read-only critic subagent with:

- role: critic
- objective: apply `<lens>` to `<target>` and find production-relevant design,
  correctness, test, or operability risks
- scope: target path, relevant nearby contracts, and the exact lens excerpt
- boundary: no edits, no broad review, no author reasoning trail
- output: structured findings only

If the harness cannot spawn a fresh subagent, the primary adopts the exact
lens text for one read-only pass and marks the header
`lens-adopted-by-primary`. This is a fallback, not fresh-context separation.

## Output

```markdown
## Critique
- Lens: <name>
- Target: <path>
- Evidence read: <files/commands inspected>
- Fresh-context mode: <subagent|lens-adopted-by-primary>
- Findings:
  - Finding: <concise claim>
    Evidence: <file:line>
    Impact: <high|medium|low>
    Suggested next step: <smallest useful action>
- Residual uncertainty: <what was not read or could change the verdict>
```

Findings require file:line evidence. If no finding survives, say so and cite
the files that were read. Do not return `Ship`, `Conditional`, or `Don't Ship`;
that verdict belongs to `/code-review`.

## Examples

- `/critique --lenses`
- `/critique --lens ousterhout --target src/auth`
- `/critique --lens cooper --target tests/auth.spec.ts`

    78	pub fn read_request(stream: &mut impl Read, scheme: &str) -> Result<Request, HttpError> {
    79	    let mut reader = BufReader::new(stream);
    80	    let mut request_line = String::new();
    81	    if reader.read_line(&mut request_line)? == 0 {
    82	        return Err(HttpError::BadRequest("empty request".to_string()));
    83	    }
    84	    let mut header_bytes = request_line.len();
    85	    let mut parts = request_line.split_whitespace();
    86	    let method = parts
    87	        .next()
    88	        .ok_or_else(|| HttpError::BadRequest("missing method".to_string()))?;
    89	    let target = parts
    90	        .next()
    91	        .ok_or_else(|| HttpError::BadRequest("missing target".to_string()))?;
    92	    let mut request = Request::new(method, target);
    93	    request.scheme = scheme.to_string();
    94	    let mut content_length = 0usize;
    95	    let mut header_lines = 0usize;
    96	    loop {
    97	        let mut line = String::new();
    98	        if reader.read_line(&mut line)? == 0 {
    99	            return Err(HttpError::BadRequest(
   100	                "headers were not terminated".to_string(),
   101	            ));
   102	        }
   103	        header_lines += 1;
   104	        header_bytes += line.len();
   105	        if header_lines > MAX_HEADER_LINES || header_bytes > MAX_HEADER_BYTES {
   106	            return Err(HttpError::BadRequest("headers too large".to_string()));
   107	        }
   108	        let line = line.trim_end_matches(['\r', '\n']);
   109	        if line.is_empty() {
   110	            break;
   111	        }
   112	        let Some((name, value)) = line.split_once(':') else {
   113	            return Err(HttpError::BadRequest("malformed header".to_string()));
   114	        };
   115	        let name = name.trim();
   116	        let value = value.trim();
   117	        if name.eq_ignore_ascii_case("host") {
   118	            request.host = value.to_string();
   119	        }
   120	        if name.eq_ignore_ascii_case("content-length") {
   121	            content_length = value
   122	                .parse::<usize>()
   123	                .map_err(|_| HttpError::BadRequest("invalid content-length".to_string()))?;
   124	            if content_length > MAX_BODY_BYTES {
   125	                return Err(HttpError::BadRequest("body too large".to_string()));
   126	            }
   127	        }
   128	        request.headers.insert(name, value);
   129	    }
   130	    if content_length > 0 {
   131	        request.body.resize(content_length, 0);
   132	        reader.read_exact(&mut request.body)?;
   133	    }
   134	    Ok(request)
   135	}

   153	pub fn is_loopback_host(host: &str) -> bool {
   154	    let without_port = host
   155	        .strip_prefix('[')
   156	        .and_then(|host| host.split_once(']').map(|(host, _)| host))
   157	        .or_else(|| host.rsplit_once(':').map(|(host, _)| host))
   158	        .unwrap_or(host);
   159	    without_port == "localhost"
   160	        || without_port
   161	            .parse::<IpAddr>()
   162	            .is_ok_and(|ip| ip.is_loopback())
   163	}

    96	    let right = right.as_bytes();
    97	    let max = left.len().max(right.len());
    98	    let mut diff = left.len() ^ right.len();

   101	        let right_byte = right.get(index).copied().unwrap_or_default();

   136	fn session_path(path: &str) -> Result<Option<SessionPath>, ()> {
   137	    let Some(rest) = path.strip_prefix("/v1/sessions/") else {
   138	        return Ok(None);
   139	    };
   140	    let rest = rest.trim_matches('/');
   141	    if rest.is_empty() {
   142	        return Ok(None);
   143	    }
   144	    let parts = rest.split('/').collect::<Vec<_>>();
   145	    if parts.len() > 2 {
   146	        return Ok(None);
   147	    }
   148	    let key = percent_decode(parts[0]).ok_or(())?;
   149	    let action = parts.get(1).map(|part| match *part {
   150	        "ack" => SessionAction::Ack,
   151	        "stop" => SessionAction::Stop,
   152	        "turns" => SessionAction::Turns,
   153	        _ => SessionAction::Unknown,
   154	    });
   155	    if matches!(action, Some(SessionAction::Unknown)) {
   156	        return Ok(None);
   157	    }
   158	    Ok(Some(SessionPath { key, action }))
   159	}

   161	fn percent_decode(raw: &str) -> Option<String> {
   162	    let mut out = Vec::new();
   163	    let mut bytes = raw.as_bytes().iter().copied();
   164	    while let Some(byte) = bytes.next() {
   165	        if byte == b'%' {
   166	            let high = bytes.next()?;
   167	            let low = bytes.next()?;
   168	            let value = (hex_value(high)? << 4) | hex_value(low)?;
   169	            out.push(value);
   170	        } else {
   171	            out.push(byte);
   172	        }
   173	    }
   174	    String::from_utf8(out).ok()
   175	}
   176
   177	fn hex_value(byte: u8) -> Option<u8> {
   178	    match byte {
   179	        b'0'..=b'9' => Some(byte - b'0'),
   180	        b'a'..=b'f' => Some(byte - b'a' + 10),
   181	        b'A'..=b'F' => Some(byte - b'A' + 10),
   182	        _ => None,
   183	    }
   184	}

     8	    let state_dir = state_dir.as_ref();
     9	    let path = state_dir.join("api.token");

    15	                return Err(ApiError::Config("api token file is empty".to_string()));
    16	            }
    17	            Ok((token, path))
    18	        }
    19	        Err(error) if error.kind() == std::io::ErrorKind::NotFound => {
    20	            fs::create_dir_all(state_dir)
    21	                .map_err(|source| ApiError::Internal(format!("create state dir: {source}")))?;
    22	            set_dir_private(state_dir)?;
    23	            let mut raw = [0u8; 32];
    24	            getrandom::fill(&mut raw)
    25	                .map_err(|source| ApiError::Internal(format!("generate api token: {source}")))?;
    26	            let token = hex::encode(raw);
    27	            write_new_private_file(&path, format!("{token}\n").as_bytes())?;
    28	            Ok((token, path))
    29	        }
    30	        Err(error) => Err(ApiError::Internal(format!("read api token: {error}"))),
    31	    }
    32	}
