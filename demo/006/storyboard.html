Curb Demo Storyboard

Controlled walkthrough using a synthetic worker, not live model sessions.

Observe

Curb starts in visibility mode

Run curb validate-config and curb scan --json against the isolated demo config.

Warn

A synthetic worker crosses the warning line

Alert mode records and surfaces a warning while preserving the invariant that it never terminates processes.

Acknowledge

The operator extends the run

The acknowledgement is explicit, bounded, and visible in the evidence ledger.

Enforce

Only the controlled worker is stopped

Curb revalidates PID and start-time identity before terminating the synthetic process tree.

Privacy

Evidence without content capture

Curb records process and usage metadata, policy events, and ledger hashes. It does not record prompts, responses, screenshots, keystrokes, or file contents.
